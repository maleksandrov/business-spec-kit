## Non-Negotiables

<!-- Things that MUST always appear in outputs -->
- Always include a clear call-to-action in external-facing documents
- Always quantify claims where supporting data exists
- Always name the specific problem before proposing a solution

<!-- Things that must NEVER appear in outputs -->
- Never use the phrase "best-in-class"
- Never promise outcomes not supported by the brief
- Never include pricing in early outreach without explicit instruction from the brief

## Preferred Formats

- Use bullet points for risk lists, objection maps, and feature comparisons
- Use numbered lists for sequential steps and ranked priorities
- Use tables for comparisons, timelines, and task lists
- Use **bold** to highlight key terms on first use in a document
- Keep executive summaries to 3–5 sentences maximum
- Avoid nested bullet points deeper than two levels
